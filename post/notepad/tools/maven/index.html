maven

以下的命令会按顺序执行，后面命令会调用前面的命令

1

mvn clean

把编译好的类清除掉， 因为每个开发环境不同

1

mvn compile

编译主要的类

1

mvn test

编译测试的类

1

mvn package

把工程打包成jar 包

编译测试类

1

mvn install

执行上面的所有命令， 并且放入本地仓库中

maven工程可以直接使用：

1

tomcat:run

直接使用tomcat 运行项目

windows安装：

首先， 下载可执行文件：http://maven.apache.org/download.cgi

下载后解压记住路径， 然后添加系统变量和环境变量。

最后测试：

1

mvn -version

本地仓库: 使用时先在本地找依赖， 如果没有就去网络找！网络找到后会放在这里
